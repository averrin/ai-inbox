# Calendar Configuration Delta

## Changed Requirements

-   **Calendar Grouping**: Calendars MUST be categorized into "Personal", "Work", or "Additional".
    -   "Personal" and "Work" sections MUST allow selecting multiple calendars.
    -   "Additional" section contains all other calendars not explicitly assigned to Personal or Work.
-   **Account Assignment**:
    -   The "Work" section MUST allow selecting a specific email/account to be used for work invites.
