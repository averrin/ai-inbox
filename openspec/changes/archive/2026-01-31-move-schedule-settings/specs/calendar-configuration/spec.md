## ADDED Requirements

### Requirement: Calendars Settings Screen
The system SHALL provide a dedicated screen for configuring calendar connections and preferences.

#### Scenario: Accessing Calendars Settings
- **WHEN** user navigates to the root Settings tab
- **THEN** a "Calendars" item is visible
- **WHEN** user taps "Calendars"
- **THEN** the Calendars Settings screen is displayed

#### Scenario: Calendar Content
- **WHEN** viewing the Calendars Settings screen
- **THEN** user can view and manage connected calendars (toggle visibility, set default, etc.)
