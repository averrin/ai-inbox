## ADDED Requirements

### Requirement: Event Types Settings Screen
The system SHALL provide a dedicated screen for managing event types.

#### Scenario: Accessing Event Types Settings
- **WHEN** user navigates to the root Settings tab
- **THEN** an "Event Types" item is visible
- **WHEN** user taps "Event Types"
- **THEN** the Event Types Settings screen is displayed

#### Scenario: Event Types Content
- **WHEN** viewing the Event Types Settings screen
- **THEN** user can list, create, edit, or delete event types
